# Locus - An All in One Student Hub

Locus is a modern student productivity platform designed to simplify academic life by bringing essential student tools into one place.

## Features

* Attendance Tracking
* Timetable Management
* Notes Organization
* Assignment Tracking
* CGPA Monitoring
* Student Dashboard Overview
* Responsive Landing Page Design

## Technologies Used

* HTML5
* CSS3
* Git
* GitHub Pages

## Purpose

Locus aims to serve as a centralized hub for students by integrating multiple academic tools into a single, intuitive platform. The landing page showcases the concept through a modern user interface featuring a dashboard preview, glassmorphism design elements, and an immersive video background.
